# 日本政策金融公庫（JFC）制度別：必要書類マトリクス（必須／推奨／参考）

## 0. 使い方
プロジェクト作成で `selected_jfc_loan_program` が確定したら、下記の「書類プロファイル」を読み込み、Document Checker に以下3レベルで表示する。

- **必須**：提出がないと申込が進まない可能性が高いもの
- **推奨**：審査・ヒアリングがスムーズになりやすいもの（不足だと追加依頼が出やすい）
- **参考**：あると説明力が上がる・補強できるもの

> 注意：制度や申込形態（窓口/オンライン）、個別状況により追加提出が求められる場合がある。最終的には公庫の案内・支店の指示に従う。

## 1. 共通（国民生活事業の事業資金：ベースセット）

### 1.1 必須（ベース）
- **創業計画書** または **企業概要書**
  - これから事業開始／開始後間もない → 創業計画書
  - それ以外 → 企業概要書
- **直近期＋前期の確定申告書（2期分）／決算書（法人は決算書含む）**
- **本人確認書類**（代表者）
- **送金先口座の確認資料**（通帳見開き等）

### 1.2 推奨（ベース）
- **見積書**（設備資金がある場合）
- **試算表**（決算後6か月以上経過 or まだ決算未了の場合）
- **許認可証等**（許可・届出が必要な業種の場合）
- **勘定科目明細**（決算書一式に含まれることが多い）

### 1.3 参考（ベース）
- 資金繰り表（月次）
- 売上・粗利の根拠資料（受注見込み、契約書、見積積上げ等）
- 事業計画の補足資料（市場/競合、価格表、サービス説明資料）
- 代表者経歴資料（職務経歴書、体制図）
- 主要取引先一覧、仕入先一覧

## 2. 制度別プロファイル（差分ルール）
以下は「共通ベース」に **追加／優先度変更** があるものを記載する。

### 2.1 創業系：新規開業・スタートアップ支援資金
**想定タグ**：創業・創業後間もない方

- 必須（追加/強化）
  - **創業計画書**（必須扱い固定）
  - 資金使途の内訳（設備/運転の明細）
- 推奨
  - 自己資金の根拠（預金残高、入金履歴）
  - 収支計画（月次12か月以上）
- 参考
  - 事業の差別化資料、プロトタイプ/実績、顧客ヒアリング結果

### 2.2 生活衛生関係営業（生活衛生貸付：一般貸付など）
**想定タグ**：生活衛生関係営業

- 必須（条件付き）
  - **都道府県知事の「推せん書」**（条件により不要となる場合あり）
- 推奨
  - 許認可証（飲食店営業許可など、該当業種）
- 参考
  - 店舗平面図、設備一覧、衛生関連の運用計画

### 2.3 生活衛生：振興事業貸付（組合員向け）
**想定タグ**：生活衛生関係営業 / 経営指導 / 生産性向上

- 必須（追加）
  - **振興事業に係る資金証明書**（振興計画認定組合の組合員）
- 推奨
  - （併用希望時）**振興事業促進支援融資制度に係る事業計画書**（生産性向上計画を含む場合あり）
- 参考
  - 組合の認定情報、指導内容メモ

### 2.4 財務体質の強化：挑戦支援資本強化特別貸付（資本性ローン）
**想定タグ**：財務体質の強化（資本性ローン）

- 必須（追加/強化）
  - 事業計画（中期：2〜5年）
  - 直近の財務状況説明資料（赤字要因/改善策、借入一覧）
- 推奨
  - KPI計画（売上、粗利、人員など）
- 参考
  - 投資家向け資料（ピッチデック）、第三者評価（採択/受賞/特許等）

### 2.5 セーフティネット（売上減少・災害・感染症等）
**想定タグ**：セーフティネット / 災害関係（BCP含む）

- 必須（追加/強化）
  - 売上減少等の根拠（試算表、売上台帳、月次推移）
  - 影響の説明（原因、期間、回復見込み）
- 推奨
  - 資金繰り表（当月〜12か月）
- 参考
  - 取引先の状況資料、被災証明等（ケースにより）

### 2.6 事業再建・企業再生
**想定タグ**：事業再建・企業再生

- 必須（追加/強化）
  - 再生計画（原因分析、改善施策、返済原資）
  - 直近の試算表／資金繰り（高頻度）
- 推奨
  - 支援機関/専門家の関与資料（認定支援機関の計画、助言メモ等）
- 参考
  - 不採算部門の切り離し計画、固定費削減計画

### 2.7 事業承継・M&A / 集約・活性化
**想定タグ**：事業承継・M&A

- 必須（追加）
  - 承継計画（スキーム、相手先、時期）
- 推奨
  - 企業価値/株式評価の資料（簡易で可）
  - 引継ぎ後の経営体制資料

### 2.8 海外展開・事業再編
**想定タグ**：海外展開・事業再編

- 必須（追加）
  - 海外展開/再編の計画書（拠点、販路、リスク）
- 推奨
  - 取引契約（LOI含む）、現地パートナー資料

### 2.9 ソーシャルビジネス
**想定タグ**：ソーシャルビジネス

- 必須（追加/強化）
  - 社会課題と解決手段の整理（事業計画内でOK）
- 推奨
  - インパクト指標、協力団体の資料
- 参考
  - 助成金/委託事業の採択実績等

## 3. Document Checker への落とし込みルール（実装）

### 3.1 書類カテゴリ（例）
- 申込書類
- 計画書類（創業計画書/企業概要書/各種事業計画書）
- 財務書類（確定申告書/決算書/試算表）
- 資金使途（見積書/契約書）
- 本人・法人確認（本人確認/登記/許認可）
- 補足資料（資金繰り/売上根拠/体制 等）

### 3.2 優先度の自動調整（条件付き）
- `equipment_funding = true` → 見積書を「必須」へ繰り上げ
- `tax_filing_terms < 2` → 提出必須数を調整（未了の場合は不要扱い）
- `legal_permit_required = true` → 許認可証等を「必須」または「推奨」へ
- `life_hygiene = true` かつ `loan_program = 生活衛生貸付/振興事業貸付` → 推せん書/資金証明書を要求
